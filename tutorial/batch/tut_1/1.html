Now that we are done with all the "How this tutorial works" stuff, we can get started.
Batch is a scripting or programming language, that only works on Windows Machines. It is heavialy based on the terminal,
and has some similarities with MS-DOS.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

We will now make a calculator. To create a batch file, you need to open a text document (Right-Click > New > Text Document), open the file, click File > Save As and save it as calculator.bat

Make sure you change the file type from TXT to All Files. Also make sure to save the file in an empty folder.

To start editing the code, open the editor by right-clicking on your file and then clicking on the edit button.
The first line of code for any batch file, is the to diable debug output.

1. | @echo off
2. |

We can now start by writing our user input code. We first need to tell the user what to do. We can use an 'echo' tag for that.
This is the code:

1. | @echo off
2. |
3. | echo Enter the numbers and operators you want to use.

Now we need to store the user input in a variable. We can use an 'set /p' tag for that. The syntax for the 'set /p' tag is as follows:
set /p [variable name]=

1. | @echo off
2. |
3. | echo Enter the numbers and operators you want to use.
4. | set /p input=

We have the input stored in the variable 'input' now. We want to calculate what the user has given us, into a single number.
For that we use the 'set /a' tag, where a stands for arimathic.
The syntax is simple: set /a [output_variable]=[input_variable]

1. | @echo off
2. |
3. | echo Enter the numbers and operators you want to use.
4. | set /p input=
5. | set /a output=%input%

Now that we have done all of the calculating, we are ready to show the answer to the user.
For that we use the 'echo' tag again, and also include a simple %% variable.

1. | @echo off
2. |
3. | echo Enter the numbers and operators you want to use.
4. | set /p input=
5. | set /a output=%input%
6. | echo Answer: %output%

We also want to put the results and calculation into a text document.
For that we use the 'echo' tag, in combination with a ">>" (write to file) tag.
The syntax is: echo [to_be_saved_text] >>[document_name].[file_extension]
We also use a pause tag after this, so the window does not close immediatly.

Make sure your .bat file is inside of an empty folder, to make sure it does not override other files. 1. | @echo off
2. |
3. | echo Enter the numbers and operators you want to use.
4. | set /p input=
5. | set /a output=%input%
6. | echo Answer: %output%
7. | echo %time%_%date%: %input=%%output% >>calculator_outputs.txt
8. | pause
