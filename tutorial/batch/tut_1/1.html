Batch is a scripting or programming language, that only works on Windows Machines. It is heavialy based on the terminal,
and has some similarities with MS-DOS.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

We will now make a calculator. To create a batch file, you need to open a text document (Right-Click > New > Text Document), and save it as calculator.bat

Make sure you change the file type from TXT to All Files. Also make sure to save the file in an empty folder.

To start editing the code, open the editor by right-clicking on your file. Now click on the edit button.
The first line of code for any batch file, is the to diable debug output.
