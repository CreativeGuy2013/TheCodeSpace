Ok, so let's get the basic HTML structure out of the way.
First we will need a Home page for our website. Let's make one by doing the following:
Right-Click > New Folder > Rename it to the name of your website > Right-Click > New File > Rename it to be index.html

Be sure to replace the .txt extension with .html! Else it won't work!

So what we just did, is declare the file. When you open this in a web browser, you'll just get a blank screen.
The browser will read from the DOCTYPE element, so it knows which HTML version the page is made with.
When you just put 'html' in there, the browser will detect it as the newest HTML version, which currently is HTML5. And then the element declares that everything inside of it, is read like an HTML element.
